# Prerequisites 

## Ollama

- Install: https://github.com/ollama/ollama/blob/main/docs/linux.md
    
        
        curl -fsSL https://ollama.com/install.sh | sh
        

- Run Gemma2: https://ollama.com/library/gemma2
    
        
        ollama pull gemma2:2b
